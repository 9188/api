安装

安装

$ npm install -g 9188

如果 npm 卡主了 可以考虑 使用淘宝定制的 cnpm 命令行工具代替默认的 npm:

$ npm install -g cnpm --registry=https://registry.npm.taobao.org
$ cnpm install -g 9188
